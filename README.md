# StayHub – Property Rental Platform

StayHub is a full-stack property rental platform similar to Airbnb where users can browse listings, add properties, and leave reviews.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap

## Features

• User authentication (login / signup)  
• CRUD operations for property listings  
• Review and rating system  
• Category-based filtering  
• Image upload support  
• Cloud deployment

## Project Structure

controller/ – application logic  
models/ – database schemas  
routes/ – API routes  
views/ – EJS templates  
public/ – static files (CSS, images)
